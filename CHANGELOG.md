# Changelog

All notable changes to this knowledge base will be documented in this file.

## [1.0.0] - 2026-05-08

### Added
- 初始版本发布
- 流行病学数据（epidemiology.json）：12条三元组
  - 2022年中国发病/死亡数据
  - 病理类型分布（乳头状癌85%、滤泡癌10%、髓样癌5%、未分化癌<2%）
  - 发病率上升原因、性别分布、预后特征
- 分期与生物标志物（biomarkers.json）：14条三元组
  - AJCC第8版TNM分期（T1-T4b）
  - ATA复发风险分层（低危/中危/高危）
  - BRAF V600E、RET、RAS基因突变
  - Tg、TGAb、降钙素等肿瘤标志物
- CSCO 2024指南推荐（csco_2024.json）：12条三元组
  - 低危/中高危手术推荐
  - 碘131治疗适应证与禁忌证
  - TSH抑制治疗目标
  - 晚期靶向治疗（仑伐替尼/索拉非尼/Selpercatinib）
- 治疗方案（treatment.json）：13条三元组
  - 手术方式选择与适应证
  - 中央区/侧颈区淋巴结清扫
  - 碘131清甲/清灶治疗流程
  - TSH抑制治疗
  - 随访监测策略与动态风险分层
